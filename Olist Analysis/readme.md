# Olist Sales & Revenue Analysis

## Project Overview
This project analyzes the **Olist e-commerce dataset** to provide insights on sales performance, revenue trends, and order fulfillment. The goal is to understand customer behavior, product performance, and overall business health through data cleaning, exploratory data analysis (EDA), and visualizations.

## Dataset
The project uses the **Olist dataset** (available on [Kaggle](https://www.kaggle.com/olistbr/brazilian-ecommerce)) which contains multiple tables:

- **orders** – information about each order (status, purchase date, delivery date, etc.)
- **order_items** – details about products in each order
- **products** – product details (category, name, price)
- **sellers** – seller information
- **customers** – customer details
- **payments** – payment information

The dataset includes **5+ CSV tables** that were merged and cleaned for analysis.

## Objectives
- Perform **data cleaning** and preprocessing to handle missing values, duplicates, and inconsistent data
- Conduct **Exploratory Data Analysis (EDA)** to identify trends in sales and revenue
- Visualize key metrics like:
  - Total sales and revenue by month
  - Order status distribution
  - Top-selling products
  - Revenue per customer region
- Create an interactive **Power BI dashboard** to present insights clearly

## Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn) – for data cleaning and EDA
- **SQL** – for querying and merging multiple tables
- **Power BI** – for dashboard creation and visualization

## Key Insights
- **Order Status**: ~90% of orders were delivered successfully, indicating smooth fulfillment
- **Revenue Trends**: Peaks in sales observed during [Dec 2017 to nov 2018]
- **Top Products**
- **Customer Distribution**
> The full dashboard provides interactive filters for better insight into sales and revenue

## Dashboard
- A clean and simple **Power BI dashboard** was created to summarize sales and revenue trends
- Features include:
  - Monthly revenue trends
  - Order status visualization
  - Top products and categories
  - Customer region analysis
